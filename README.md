# WSProject

## Introduction

This folder contains a Android project developed by the developer Leonardo Biondo Bai.

## Project notes

This projects uses Kotlin as the main language, MVVM, Clean Architeture and some top notch libraries, for example:

1. **JetPack Compose**
2. **Hilt**
3. **Coroutines**
4. **FirebaseAuth**
5. **FirebaseDatabase**
6. **Room**
7. **JUnit**
8. **Espresso**
